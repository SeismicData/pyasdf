This is the documentation of the Python API for the Adaptable Seismic Data Format (ASDF). For further information and contact information please see these two web sites:

  • Landing page of the ASDF data format: http://seismic-data.org

  • Github repository of pyasdf: http://www.github.com/SeismicData/pyasdf

We gratefully acknowledge support from the EU-FP7 725 690 VERCE project (number 283543, www.verce.eu).

Note

This is the Adaptable Seismic Data Format - if you are looking for the Advanced Scientific Data Format, go here: https://asdf.readthedocs.io/en/latest/
